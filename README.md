# new-music-scraper
Running python script at 23:58 every night to pull in music played on the following radio stations: 
- BBC Radio 6


This script is an automated script that runs daily. It takes a list of all songs played on the above radio stations and produced a table within Obsidian. As Deezer is not accepting new apps, a csv. file is created once a week. The user then can manually upload this to the music platform of their choice to create a playlist of the songs.
